This repository is linked to the following research paper for peer-review: Sirisha Kalidindi, Gabriel Abrahão, Anna B. Harper, Michael Crawford, Jens Heinke, Elmar Kriegler, Joeri Rogelj (2026). Quantifying uncertainty in land-based carbon removals in AR6 mitigation pathways.
The repository includes the Python scripts to reproduce Figures and the analysis in the paper.
